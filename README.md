## **How to use**
Navigate to `game` folder. From there you can run the following commands:
* ```bash 
  ~$ make
  ```
  Compiles and runs the game. Automatically removes .o and .exe files after execution is complete.

* ```bash 
  ~$ make clean
  ```
  Removes all .o and .exe files.
